# data-analytics-dashboards
This repository showcases my Power BI projects created using real-world datasets.   All data cleaning and transformation were performed using SQL and Power Query.  

Key dashboards included:

### 1️⃣ HR Analytics Dashboard
- Tools: Power BI, DAX, SQL
- Key KPIs: Attrition Rate, Average Age, Salary, Departmental Distribution
- Dataset: Kaggle (HR Employee Attrition)

### 2️⃣ Coffee Sales Dashboard
- Tools: Power BI, SQL
- Key Metrics: Total Sales, Transactions, Time-based Analysis
- Dataset: Kaggle Coffee Shop Sales

### 3️⃣ Finance Sales Performance Dashboard
- Tools: Power BI, DAX
- Key KPIs: Sales Actual vs Target, Variance %, YTD Metrics
- Dataset: Finance KPIs (Cleaned & Processed)
